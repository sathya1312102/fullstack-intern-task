# fullstack-intern-task

Mini SaaS Template Store built with React, Node.js, Express, and MongoDB.

## Tech Stack

- Frontend: React.js (Vite) + TailwindCSS
- Backend: Node.js + Express.js
- Database: MongoDB
- Auth: JWT + bcryptjs

## How to Run

### Server

cd server
npm install
npm run dev

### Client

cd client
npm install
npm run dev

## Seed Database

cd server
node seed/seedTemplates.js

## Features

- Register and Login
- View templates with search and category filter
- Add/remove favorites
- My Favorites page (protected)
- Responsive design
